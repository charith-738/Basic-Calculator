# Calculator Project

This is a basic calculator built with HTML, CSS, and JavaScript.

## Features:
- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Clear button to reset the calculator.
- Equals button to compute the result.

## How to Use:
1. Click on the buttons for digits and operators.
2. Click the "=" button to calculate the result.
3. Click the "C" button to clear the display.
